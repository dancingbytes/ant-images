# Ant::Images

Support images for Ant editor

## Support

Ruby:  2.0+

Rails: 4.2+

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ant-images'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ant-images

## Usage

Add to application.js

```js
//= require ant/images
```

Add to application.scss

```css
@import "ant/images";
```
